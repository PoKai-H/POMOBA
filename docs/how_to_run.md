### How to run the agent:

 1. Pull the latest changes
 2. Open Godot
 3. Open terminal in root directory
 4. Activate the Python environment called rl using

``` 
source rl/Scripts/activate
```

 5. Run the python file

``` 
python simple_ai.py 
```
6. Make sure the Sync node in Godot is not set to human

 7. Press play in the Godot editor
